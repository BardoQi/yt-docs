# 产品特性
>【把技术当艺术，把产品当作品】是我们的准则。我们力争为用户提供有用且好用的产品。

## 智能获客
   
>蒲公英数字化营销系统提供以下工具让您方便，轻松，简单地获取客户。 
   
* 内容分享
  
你可以采集微信中的热点文章，然后，转发给你的朋友。同样，你也可以采集热点视频，分享给你的朋友。
  
* 名片分享
  
蒲公英数字化营销系统提供智能数字化名片，你可以分享名片页面，也可以分享名片海报。并且，因为，名片海报是图片，所以，它也可以分享到朋友圈。
  
* 动态分享
  
可以分享个人动态。而在动态中，你可以晒你的订单，做商品介绍，或者添加一些有趣的内容。
  
* 商品分享
 
虽然说，你只是一个分销员，分销是其他公司的商品，但是，你一样可以将你的商品加为推荐商品，从而分享给你的好友。同时，你也可以生成商品海报，分享给你的好友。
   
## 人脉分销
   
除了手机端操作，你一样可以利用你的线下人脉，不只是亲人，还有朋友，只在是信任你的，认识你的人，你都可以分享你的所购的商品的体验，分享你做分销员的感受。从而进一步增加你新零售门户的流量。
   
## 门店共享
   
你不是在孤军奋战。 蒲公英数字化营销平台拥有近千家线下店铺，所以，如果你远在天涯海角的亲友想上门自提，他也可以选择线下就近的店铺。因为，在平台中，所有门店是共享的。  
   
## 货源共享
   
蒲公英数字化营销平台目前拥有数百个商家，并且，预计不到一年，商家数量会突破上万个。而所有这些商家的商品，你都是可以分销的。因为，商家间的货源是共享的。所以，你可以选择分销你熟悉的商品。
